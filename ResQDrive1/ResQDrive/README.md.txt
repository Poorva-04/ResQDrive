# ResQDrive - An OnRoad Vehicle Breakdown assistance system.

ResQDrive is a web app that provides on-road vehicle breakdown assistance.  
Users can register, book mechanic appointments, and track service status.

## Features

- User registration and login  
- Book and track appointments with nearby mechanics  
- Mechanic dashboard to manage service requests  
- Admin dashboard for system management

## Technologies

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: SQLite3

## How to Run

1. Clone the repo  
2. Install dependencies with `npm install`  
3. Run the server using `node server.js`  
4. Open the frontend in your browser
